# UAV Algorithms Internship Project

This repository contains the implementation of core algorithms developed during my internship at the Committee of Digital Assets and Breakthrough Technologies (Republic of Kazakhstan).

## Implemented Algorithms

### 1. A* Path Planning
- Grid-based pathfinding
- Supports no-fly zones
- Uses Euclidean heuristic

### 2. UAV Selection Algorithm
- Multi-criteria decision-making (WSM)
- Considers range, payload, cost, wind, and mission constraints

### 3. Particle Swarm Optimization (PSO)
- Optimizes UAV routes
- Minimizes path length using intermediate waypoints

## Technologies
- Python 3.11
- NumPy
